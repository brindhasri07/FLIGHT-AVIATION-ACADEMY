# **Flight Aviation Academy Website ✈️**

This is a simple HTML-based website created to showcase a Flight Aviation Academy and its courses. The project includes a home page with academy details, an embedded aviation video, course information, and a registration form for students who want to enroll.

## Features

Academy introduction and description
Embedded YouTube aviation video
Course categories and aviation technologies
Student registration form
Basic navigation between pages

## Technologies Used :
HTML5

## Pages

index.html – Home page

register.html – Registration form page

This project is suitable for beginners learning HTML and basic web page structure.
